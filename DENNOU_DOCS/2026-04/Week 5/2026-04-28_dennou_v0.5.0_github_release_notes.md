# DennouAibou v0.5.0

Upstream base: OpenClaw 2026.4.5

Git tag: `dennou-v0.5.0`

## Highlights

- First GitHub release using DennouAibou's own release identity.
- Keeps OpenClaw as the upstream base while separating DennouAibou versioning from OpenClaw's date-based releases.
- Adds DennouAibou-specific session pruning, protection rules, and WebUI configuration under the `dennou` config namespace.
- Continues the hard-fork lightweighting direction by removing or disabling unused upstream bundles and repairing the follow-up build/test drift.

## DennouAibou changes

### [SOUL]

- Added `src/dennou-soul/` as the isolated home for DennouAibou-specific runtime behavior.
- Added closed-session tool output pruning.
- Added active-session idle pruning with timer cancellation and atomic rewrite safeguards.
- Added prune protection for important workspace files and keywords such as `AGENTS.md`, `SOUL.md`, and `DENNOU_RULES`.
- Integrated DennouAibou settings into `openclaw.json` under the `dennou` namespace.
- Added a Control UI `DennouAibou` settings section through a tracked patch.

### [DEBLOAT]

- Removed unused native app bundles and unused bundled integrations according to the hard-fork lightweighting plan.
- Added compatibility shims for removed plugin SDK facades so build output can still be generated cleanly.
- Updated debloat documentation with the remaining regression fixes and build validation notes.

### [FIX-UPSTREAM]

- Added a safe session-store after-save hook for DennouAibou maintenance work.
- Fixed missed hook execution on the session-store ENOENT retry path.
- Preserved upstream behavior by keeping DennouAibou logic behind a narrow hook boundary.

## Configuration

Minimal example:

```json
{
  "dennou": {
    "toolsPrune": {
      "minPrunableToolChars": 1200,
      "keepLastTools": 5,
      "dryRun": true
    },
    "sessionToolsPrune": {
      "enabled": false
    },
    "activeSessionToolsPrune": {
      "enabled": true,
      "idleDelayMinutes": 30,
      "keepLastTools": 10
    },
    "pruneProtection": {
      "protectedContentKeywords": ["AGENTS.md", "SOUL.md", "DENNOU_RULES"]
    }
  }
}
```

Defaults are safe: pruning stays dry-run unless explicitly changed.

## Release scope

This release is a GitHub release only.

npm publishing is intentionally deferred to a later phase because DennouAibou still carries OpenClaw package identity and many `openclaw/plugin-sdk/*` references. Publishing under npm safely needs a separate package-name and SDK-alias migration plan.
