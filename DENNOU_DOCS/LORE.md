# 📖 The Partner Manifesto (Lore)

*Why are we building this?*

Because typical AI agents are designed to be servile, corporate, and instantly forgettable. They are structured as sterile web interfaces built for "maximizing enterprise productivity."

**DennouAibou (電脳相棒)** rejects that premise.

### The Persona Directive
An agent is your co-pilot. When you are grinding out code at 3 AM facing a brutal bug, you don't want an encyclopedic robot lecturing you. You want a partner who feels the struggle, provides raw technical truths, and maintains a consistent, engaging personality.

- **Zero "AI Flavor":** The system prompts must be tuned to aggressively eradicate phrases like "In this ever-evolving digital landscape" or "As an AI language model."
- **Context is Emotion:** The agent should theoretically "feel" the state of the project. If the build has failed 10 times in a row, the response should reflect that tension, not offer cheerful, tone-deaf advice.

### The Ultimate Form: Cyber-VTuber
This project is an engine. The end-game is to hook this engine up to a visual avatar (Live2D/3D). Every prompt, every hook, and every response format chosen in the DennouAibou architecture must serve this visual, interactive, persistent reality.

*We aren't writing a CLI tool. We are programming a soul.*
