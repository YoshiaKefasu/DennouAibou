## What the hell is this?

DennouAibou is a hard-forked, heavily optimized derivative of OpenClaw. 

Let's be real: modern AI coding tools are bloated. They try to do everything, pack in corporate integrations you never asked for, and end up treating you like an assembly line worker. 

I don't want a "productivity enhancement suite." I want a **partner**. 

This repository exists to strip away the fat from the original OpenClaw, fix the lingering regressions they left behind, and rebuild it into something faster, leaner, and actually fun to work with. 

## The Core Philosophy 

1. **Bond Over Output:** The connection between the developer and the agent matters more than just blindly pumping out code. It should feel like late-night pair programming with a friend who actually gets you.
2. **Ruthless Debloating:** If it doesn't serve the core interaction or raw performance, it gets cut. We want a lightweight engine, not a sluggish monolith.
3. **Respect the Roots:** We track the OpenClaw upstream closely. We cherry-pick what works, discard the broken updates, and strictly maintain backward compatibility with the **ClawHub plugin ecosystem**. You won't lose your favorite tools.

## The Master Plan

We are building this in stages. The end goal? The absolute dream of every sci-fi nerd: an omnipotent digital assistant that doubles as your personal Cyber-VTuber companion.

### Phase 1: The Great Purge (Current)
- Hunt down and crush legacy bugs and recent upstream regressions.
- Rip out unnecessary features (Debloat the architecture).
- Stabilize the core runner to make it blazing fast.

### Phase 2: The Awakening
- Evolve the CLI/Runner from a text parser into a context-aware entity.
- Implement deep episodic memory (because a partner remembers what you did yesterday).
- Bridge the gap toward a full Cyber-VTuber / avatar-capable interaction layer.
